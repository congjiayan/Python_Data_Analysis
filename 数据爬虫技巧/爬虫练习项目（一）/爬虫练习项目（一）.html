爬虫练习项目（一）
  • 案例一：豆瓣图书数据采集（1）
    • 任意一图书类别的书籍信息数据
      • 参考网址：https://book.douban.com/tag/%E7%94%B5%E5%BD%B1
    • 爬虫逻辑：【分页网页url采集】-【数据信息网页url采集】-【数据采集】
    • 要求
      • 1）函数式编程
        • 函数1：get_urls(n) → 【分页网页url采集】
          • n：页数参数
          • 结果：得到一个分页网页的list
        • 函数2：get_dataurls(ui,d_h,d_c) → 【数据信息网页url采集】
          • ui：分页网址
          • d_h：user-agent信息
          • d_c：cookies信息
          • 结果：得到一个数据信息网页的list
        • 函数3：get_data(ui,d_h,d_c) → 【数据采集】
          • ui：数据信息网页
          • d_h：user-agent信息
          • d_c：cookies信息
          • 结果：得到数据的list，每条数据用dict存储
      • 2）采集字段
        • 总共4个部分，其中②整体采集下来
      • 3）设置登录信息cookie
      • 4）注意添加异常处理：try...except...
      • 5）采集200条数据
        • 10页【分页网页url采集】- 每页20条数据
      • 6）结果保存为dataframe，并做基本数据清洗
        • 评分 字段
        • 页数 字段
        • 评价数量 字段
    • 通过本次案例学习什么？
      • ① 函数式编程的思路
      • ② 爬虫逻辑【分页网页url采集】-【数据信息网页url采集】-【数据采集】下的采集方法
      • ③ 哪些信息需要通过访问网页得到？哪些信息可以直接生成？→ 尽量减少访问网页次数
      • ④ 数据采集与数据清洗分开做
      • ⑤ 代码分成两部分书写，通过if __name__ == "__main__" 分开
        • 第一部分：构建函数
        • 第二部分：定义参数，并执行函数实现代码
        • if __name__ == "__main__"的意义是什么？
          • 一个python的文件有两种使用的方法，第一是直接作为脚本执行，第二是import到其他的python脚本中被调用（模块重用）执行。因此if __name__ == 'main': 的作用就是控制这两种情况执行代码的过程，在if __name__ == 'main': 下的代码只有在第一种情况下（即文件作为脚本直接执行）才会被执行，而import到其他脚本中是不会被执行的
    • 本次采集难点
      • 详细信息这里，由于部分文字包含链接-a标签，且无法固定，例如有些作者不会有a标签，较难标准化

  • 案例四：去哪儿网景点数据采集
    • 任意一城市的景点网址
      • 参考网址：https://travel.qunar.com/p-cs299878-shanghai-jingdian-1-3
    • 爬虫逻辑：【分页网页url采集】-【数据采集】
    • 要求
      • 1）函数式编程
        • 函数1：get_urls(city,n) → 【分页网页url采集】
          • city：城市对应的编码
          • n：页数参数
          • 结果：得到一个分页网页的list
        • 函数2：get_data(ui,d_h,d_c) → 【数据采集】
          • ui：数据信息网页
          • d_h：user-agent信息
          • d_c：cookies信息
          • 结果：得到数据的list，每条数据用dict存储
      • 2）采集字段
        • 景点名称、评分、排名、简介、攻略提到数量、点评数量、多少比例驴友来过、经纬度
      • 3）设置登录信息cookie
      • 4）注意添加异常处理：try...except...
      • 5）采集200条数据
        • 20页【分页网页url采集】- 每页10条数据
      • 6）结果保存为dataframe，并做基本数据清洗
        • 经度、纬度、攻略提到数量、点评数量 → 数据格式转换
        • 评分
        • 排名
        • 多少比例驴友来过
    • 通过本次案例学习什么？
      • ① 学会设定多个参数，使得函数功能更强 - 设定城市名
      • ② 从图示信息中采集数据，例如这里的评分
      • ③ 采集经纬度
